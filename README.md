# Todo List App (Vue 3 + TypeScript)

Лабораторная работа №8: перенос проекта на TypeScript, настройка ESLint и Prettier.

## Требования к выполнению работы
Перенести существующий проект To‑Do List, выполненный ранее на JavaScript, полностью на TypeScript.
Настроить в проекте ESLint с рекомендованными правилами для Vue 3 + TypeScript.
Настроить Prettier как форматтер кода и интегрировать его с ESLint.
Привести весь проект к единому стилю кода согласно настройкам линтера и форматтера.
Обеспечить отсутствие ошибок ESLint при сборке.

## Технологии
- Vue 3 
- TypeScript
- Vite
- SCSS
- Pinia
